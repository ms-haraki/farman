# farman アプリケーション仕様書

## 概要

Qt6 / C++20 製のクロスプラットフォーム2画面ファイラ。キーボード操作を主軸とし、
Total Commander / Double Commander に近い操作感を目指す。

---

## 対応プラットフォーム

- macOS
- Windows
- Linux

---

## ペイン

### ファイル一覧の表示列

| 列 | 内容 |
|---|---|
| 名前 | ファイル名・ディレクトリ名 |
| サイズ | ファイルサイズ（ディレクトリは空欄 or アイテム数） |
| 種別 | 拡張子から判定した種別 |
| 更新日時 | 最終更新日時 |

- 表示列はカスタマイズ可能

### 表示オプション

- 隠しファイルの表示・非表示
- ディレクトリを先頭に表示
- `.` / `..` の表示

### ソート

- ソートキー: 名前・サイズ・種別・更新日時
- 第2ソートキーも指定可能
- 昇順・降順
- 大文字小文字の区別
- ディレクトリの表示
  - ディレクトリをリスト先頭・最後尾・特になし
- ソート基準はディレクトリ単位で記憶

### フィルタ

- 名前マスク（glob: `*.jpg *.png` など任意の文字列で複数指定可）
- 属性フィルタ（通常ファイル・ディレクトリ・隠しファイル等）
- フィルタはディレクトリ単位で記憶

## キーボード操作

### 基本ナビゲーション

| キー | 動作 |
|---|---|
| `↑` / `↓` | カーソル移動 |
| `Enter` | ディレクトリに入る / ファイルをビュアーで開く |
| `Backspace` | 親ディレクトリへ移動 |
| `Tab` | アクティブペインを切り替え |
| `Space` | カーソル行を選択・解除してカーソルを下に移動 |
| `Ins` | カーソル行を選択・解除（カーソル移動なし） |
| `*` | 選択を反転 |
| `Ctrl+A` | 全選択 |

### ファイル操作

| キー | 動作 |
|---|---|
| `c` | 選択ファイルを反対ペインへコピー |
| `m` | 選択ファイルを反対ペインへ移動 |
| `k` | 新規ディレクトリ作成 |
| `d` | 選択ファイルを削除 |
| `n` | 新規ファイル作成 |
| `r` | リネーム |
| `a` | 属性変更 |

### その他

| キー | 動作 |
|---|---|
| `1` | シングルペイン |
| `2` | デュアルペイン |
| `Ctrl+Enter` | 任意のビュアーで開く |
| `Shift+Enter` | ファイル実行 |
| `f` | ファイル検索 |
| `b` | ブックマークに追加 |
| `Ctrl+b` | ブックマーク一覧 |
| `Alt+←` / `Alt+→` | ディレクトリ履歴を前後に移動 |

※ 上記はデフォルトキーアサイン。全て変更可能。

## ファイル操作

### コピー・移動

- 複数ファイル・ディレクトリ対応
- 進捗ダイアログ（ファイル名・進捗バー・速度・残り時間）
- 上書き確認ダイアログ（はい・いいえ・全て・スキップ・キャンセル）
- バックグラウンドスレッドで実行（UIをブロックしない）

### 削除

- 確認ダイアログ
- ゴミ箱に送る / 完全削除 を選択可能

### 圧縮・解凍

- 対応フォーマット: zip・tar・tar.gz・tar.bz2・tar.xz
- 選択ファイルを圧縮
- アーカイブを展開

### リネーム

- 単一ファイルのインラインリネーム
- 一括リネーム（連番・正規表現置換）

---

## ファイル検索

- 検索ディレクトリ（サブディレクトリ含む）
- 条件: ファイル名（glob）・サイズ・更新日時・内容（テキスト検索）
- 結果を一覧表示し、そこからファイル操作可能

---

## ビュアー

- `F3` またはファイルによっては `Enter` で起動
- 拡張子・MIMEタイプから自動判別

### 組み込みビュアー

| 種別 | 対象 |
|---|---|
| テキストビュアー | .txt .log .cpp .h .json .yaml 等 |
| 画像ビュアー | .jpg .png .gif .bmp .webp 等 |
| Hexビュアー | バイナリファイル全般・フォールバック |

### プラグインシステム

- 動的ライブラリ（.dylib / .dll / .so）として実装
- インターフェース: `IViewerPlugin`
  - `supportedExtensions()` で対応拡張子を宣言
  - `createViewer()` でビュアーウィジェットを生成
- 所定のディレクトリに置くだけで自動ロード

---

## ブックマーク

- よく使うディレクトリを登録
- 名前を付けて管理
- キーで一覧を開いてジャンプ

---

## 設定

### カスタマイズ可能な項目

- キーアサイン（全コマンド）
- フォント・フォントサイズ
- カラーリング（ファイル種別ごとの色）
- 日時フォーマット
- ファイルサイズの表示形式（バイト・KB・MB・KiB・MiB・自動）
- 起動時のディレクトリ（前回のパス・固定パス・デフォルト(ユーザーディレクトリ)）
- カーソル形状（下線 or 背景色・文字色を変える）

### 設定ファイル

- `QSettings` を使用
- macOS: `~/Library/Preferences/`
- Windows: レジストリ or `%APPDATA%`

---

## 非機能要件

- 起動時間: 1秒以内
- 大量ファイル（10,000件以上）でもスクロールが滑らか
- ファイル操作はバックグラウンドスレッドで実行
